Cartoon List

In this course, we used Android Studio to develop basic applications. For our final project, we were given the freedom to make any application we wanted, so I chose to create an application that allows users to create a list of their favorite cartoons and edit the scores. It's a simple application, but I was only given about a week to create it and I'm happy with the finished product.
